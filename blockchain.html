Digital Workflows on Blockchain

Blockchain is a shared, immutable ledger that facilitates the process of recording transactions and tracking assets in a business network. It improves inter-workflow efficiency and to allow cross industry collaboration. Uplifting the construction digitisation standard

The Blockchain Network >

What is stored on blockchain?

Workflow metadata is stored on the blockchain and documents are stored on a separated, immutable file system. Blockchain captures metadata and documents in major status update in a workflow.

The Blockchain

The Blockchain is built with Ethereum Proof of Authority (PoA) consensus. A permission based/private blockchain that connects to only authorised systems and users. 

Smart Contract

Solidity based smart contract is developed to read and write content from/to the Blockchain. Web3 functions are developed to interact with smart contract functions.

Inter-Planetary File System (IPFS)

IPFS is a P2P file system that stores large documents. Documents are identified by Contents ID (CID). CID becomes invalid if content is modified. CID is stored on Blockchain to link the metadata and the related documents.
