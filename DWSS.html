We are well integrated

Our DWSS is supporting integration with these external features.
The extensive DWSS API allows any authorised external system to integrate with us rapidly.

iCWP

As Development Bureau's initiative, integrated Capital Works Platform (iCWP) requires DWSS data in XML to be pushed to the iCWP on daily basis.
This is to enable data integration and analysis of works information collected from different works projects for continuous monitoring and review of various project performance with a view to enhancing the management of the Capital Works Programme. DWSS data. 

BIM

BIM Object IDs are retrieved from a specific version of BIM model. Inspection location (i.e. in RISC) can optionally select a BIM Object to attach with.
DWSS user can click on a BIM Object to open a web BIM viewer with the object highlighted.

Smart Site Safety System (4S)

4S system dashboard is to be integrated to DWSS dashboard to provide a secondary view.
Inspection Test Plan (ITP) and Work Permits (i.e. Permit to Lift) are implemented in DWSS to safeguard site safety.
Mobile App

Android and iOS mobile applications are available for download from Google Play and Appstore.
Users can view, create, edit, delegate tasks.
Quick access to "My Action" so that you won't miss any of your tasks.
With offline mode and eSignature supported, mobility is easier and ever.
